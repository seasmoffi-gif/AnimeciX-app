### Installation

1. **Clone the repository**

```bash
git clone https://github.com/Darkx-dev/ShonenX.git
cd ShonenX
```

2. **Install dependencies**

```bash

```

3. **Run the app**

```bash

```

   - Modify `android/app/build.gradle` for Android settings
